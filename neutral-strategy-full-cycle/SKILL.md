# 中性策略全周期开发工作流 (Neutral Strategy Full Cycle Workflow)

从中性策略定性方向到最终优化方案的完整开发流程。

## 工作流步骤总览

| 步骤 | 名称 |
|------|------|
| 1 | 生成定性方向 |
| 2 | 生成单因子 |
| 3 | 遍历测试单因子分箱 |
| 4 | 生成双因子组合遍历方案 |
| 5 | 遍历测试双因子分箱 |
| 6 | 确定最终的双因子方案 |
| 7 | 遍历额外因子 |
| 8 | 生成策略优化方案 |

## §0 开发周期名称 / 开发周期文档
- **开发周期名称**：`strategy_full_cylcle` 下的**子目录名称**，也是本次全周期的全局标签，如 `中性做空2026xxxx`。该名称在后续步骤中重复引用，用于读写数据（如 §2.2 写入 FACTOR_PASSWORDS、§3.1 更新 backtest_name 等）。

- **开发周期文档**：指 `strategy_full_cylcle / {开发周期名称} / 开发周期文档.md` 这一具体 MD 文件及其全部内容（示例：`strategy_full_cylcle / 中性做空2026xxxx / 开发周期文档.md`）。

本节记录**当前激活的开发周期名称（全局标签）**。后续所有读写开发周期文档、回测名称等，都应使用该名称作为唯一来源；开启新一轮开发时，只需在此处替换为新的名称，并在 `strategy_full_cylcle/{开发周期名称}/` 下建立对应目录与文档。

## 数据源

- **SGTG 目录**：`SGTs/{开发周期名称}_{因子名}/`（如 `SGTs/中性做空2026xxxx_QvNMean/`）。热力图与汇总表位于 **SGTG 目录根**：`参键xSGT实例表现热力图.png`、`参键xSGT实例表现df.csv`。
- **SGT 子目录**：SGTG 目录下若干子目录（如 `中性做空2026xxxx_QvNMean -> (0.9, 1.1)_1255`），每个为一次完整彩虹回测，含 `SGT输入逻辑.md`、`资金曲线.csv` 等。资金曲线含全量数据，年份数据由全量截取。每个 SGT 均有正确的输入逻辑。
- **遍历范围**：识别所有以「开发周期名称」为前缀的 SGTG 目录，分析有测试结果的因子即可；不必与开发周期文档 §2 的因子列表完全一致。

## 执行流程

1. **防御性检查**：对每个因子，任选一个 SGT 子目录，读取 `SGT输入逻辑.md` 中的 `execution_mode`，与开发周期名称蕴含的多空方向（中性做空→neutral_short，中性做多→neutral_long）比对；若不一致则报错。
2. **读取图表与数据**：优先读取 SGTG 目录下的 `参键xSGT实例表现热力图.png` 观察表现；若需确认数值，参考 `参键xSGT实例表现df.csv`。子图矩阵的 x/y 轴变量名以 PNG 标注为准（分箱通常为 `pre_filter_range` 的 tuple，参数为 `param1`）；若标注不清，在输出中备注。
3. **分箱区间与四维评估**（针对每个因子）：
   - **分箱区间**：指横跨热力图矩阵 xy 轴的一个区域，由若干单格组成。例如：排名范围 0～0.2（可由 (0, 0.1)、(0.1, 0.2) 两个单格构成）、参数范围 200～600（可由 param1=200、400、600 三个单格构成）。**参数范围须满足 max≥2×min**（如 param1：100～200 可、100～150 不可）。
   - **任务**：针对每个因子，寻找 1 个或若干个**综合表现出色的分箱区间**，该区间在以下四方面均有较好表现：
     - **全量表现**：净值高、回撤低、净值回撤比高、周度收益高。

     - **分域潜力**：若全量表现一般，但该区间明显优于其他区间，可记为备选分域。
   - **目标**：基于该因子，找到若干综合表现出色的分箱区间。
4. **评估规则**：
   - **因子参数依赖**——若因子的 13 位密码中无参数配置（位3、5、7、12 均为 None，如 Cmcap：因子值即流通市值，不依赖任何窗口参数），则热力图中 param1 对因子值无影响，同一分箱下不同参数结果完全一致；此时参数稳定性与该因子无关，既非加分项也非减分项，评估时跳过该维度即可。
   - **分箱语义不可跨因子套用**——`pre_filter_range` 的语义取决于该 SGTG 的前滤因子，不同因子含义不同（如 Cmcap 的 (0, 0.1) 为小市值，TpNMinmax 的 (0, 0.1) 为价格处于相对低位）。描述分箱时不可跨因子套用语义，应使用中性表述如「因子值处于低位/高位」或「前滤因子排名靠前/靠后」。
5. **输出**：用**表格**呈现所有优质分箱区间，行按**因子整体表现**优先、其次按**分箱表现**排序（即同一因子的分箱连续排列，即使某因子的分箱1优于上一因子的分箱2，也排在后面）。表格列与格式如下：

- 7 个测试变量：因子A名称、因子A参数、因子A前滤范围、因子B名称、因子B参数、因子B选币数量、因子B选币方向。赋值时因子A参数、因子B参数对应 param1/param2。
- 第 8 个变量「参数范围」：如 param1=100～300，或 param1=100～300、param2=200～400（若有 param2）。
- 与双因子组合遍历的区别：**只有参数会变，前滤范围固定**。

- **无交集**（param1 × param2）：前滤范围固定，在二维热力图中找横跨多个参数的优质区域。

### 4. 筛选双因子组合

双因子组合须满足：
- 对应**一个单独的前滤区间**，如 (0.2, 0.5)；
- **横跨多个参数**，即对应一个参数范围，可为单 param1 范围，或 param1/param2 双参数范围（赋值在「参数范围」列）。

### 5. 评估要求

- **全量表现**：净值高、回撤（面积）低、净值回撤比高、周度收益高；
- **参数稳定**：全量数据中表现好的参数范围大；
- **年份稳定**：在全部/大部分年份中都有不错的表现。

### 6. 挑选与排序

- 数量等于 §4 输出的遍历方案数量：在 §4 的**每个**遍历方案中，各挑选 **1 个**最优质的双因子组合。
